Come creare un template per il post

di Mario Rossi - pubblicato il 25-12-2020

La prima cosa da fare è suddividere in blocchi la pagian, identificando le parti comuni da applicare a tutte le pagine

Le varie aree saranno quindi suddivise in:

  • Conyenuti di intestazione del tag <header>
  • Contenuti centrali variabili del tag <main>
  • Contenuti di fine pagina comuni, da inserire nel tag <footer>
  • Formattazione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da sezionare con i CSS.Perciò i passi saranno:

  • Progettare su carta il markup finale, assegnando ad ogni specifico testo il tag appropriato
  • Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  • Applicare gli stili CSS
  • L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale.

Diffusione dei CSS
